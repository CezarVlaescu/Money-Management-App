# 🌷 Money Bloom

Money Bloom is a mobile-first budgeting application built with Angular. It helps users manage income, expenses, savings, subscriptions and investments through a simple and modern interface.

## Features

- 50/30/20 monthly budget overview
- Expense tracking and automatic categorization
- Savings goals and progress tracking
- Calendar with adaptive daily spending allowance
- Recurring payments and subscriptions
- Savings and investment account tracking
- Supabase authentication and cloud synchronization
- Local-first storage and offline support
- Dark and light themes
- Installable Progressive Web App

## Tech Stack

Angular · TypeScript · SCSS · Supabase · Netlify · PWA

## Run locally

```bash
npm install
npm start
```

Open `http://localhost:4200`.

## Disclaimer

Money Bloom is a personal budgeting tool and does not provide financial or investment advice.

Live project: https://money-bloom.netlify.app/
